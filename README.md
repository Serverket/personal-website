## :gear: Install & Run
You'll need at least `node 16` and `bun` to compile everything up:

```
bun install

bun run dev
```

## :star2: Main features

**Frontend / UI:**
* React, React Context, React Router, Hooks and HOCs
* Tailwind CSS
* FormSubmit
* Fontello
* Vite

**Animations:**
* React-Spring (Hooks API)
* Tailwind Transitions
